## Learn React Native

This course contains all the knowledge you need to create an application with React Native.

Click here to get the Guide to Creating a React Native Apps with `@tanstack/react-query`

Most beginners to React Native face the problem that tutorials assume a lot of React knowledge and make things very complicated for beginners. This results in learners getting frustrated, dragging out the learning process over months or discouraging learners from sticking with the technology.

If you've been daunted trying to learn React Native, and got stuck, then this course is for you.

In these courses, I teach you hands on to implement the most important features for mobile development with React Native. Knowing these skills means you can confidently tackle just about any mobile app idea you want to implement.

The course is full of instruction on the fundamentals of React Native development, including foundations, data management, ecosystem, navigation, and releases.

By the end of the first module of the course, you'll already have created your first React Native project for both Android and iOS using the Expo toolset. The subsequent modules will dive into more ideas such as:

- How to implement screen navigation.
- How to use a React Native design system with `react-native-paper`.
- How to setup TypeScript to catch bug as you write code.
- How to add Google Sign In to your React Native apps.
- How to build complex screens.
- How to add bottom navigation.

You can read more details about the course over at the Creating a React Native Login website.
