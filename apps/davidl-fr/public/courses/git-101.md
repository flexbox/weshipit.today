## Commit

A **snapshot** of your work

---

<img src="http://upload.wikimedia.org/wikipedia/commons/thumb/b/b4/Vincent_Willem_van_Gogh_128.jpg/1024px-Vincent_Willem_van_Gogh_128.jpg" width="40%">

---

## Commit - How to

A **commit** is a 3-steps job.

```bash
# Check the status of your files
git status

# Select which file to add to the commit —stage modified files
git add <file_1_which_has_been_modified>
git add <file_2_which_has_been_modified>

# Take a snapshot of what is in the staging area.
git commit --message "A meaningful message about this change"
```

---

## Inspect your changes

If `git status` tells you something changed, you can inspect exactly what changed:

```bash
git diff
git diff <a_specific_file_or_folder>
```

---

## Log

Show commit history with:

```bash
git log
```

# Branching

---

## One feature `===` One branch

---

![branching_1](https://courses.davidl.fr/assets/images/git-101/branching_1.png)

```bash
git branch my-feature
```

---

2 commits happen in the `my-feature` branch

![branching_2](https://courses.davidl.fr/assets/images/git-101/branching_2.png)

```bash
git branch my-feature
git checkout my-feature
git add --all
git commit --message "better typography"
git commit --message "update dependecy"
```

---

In the meantime, one commit happened in the `master` branch

![branching_3](https://courses.davidl.fr/assets/images/git-101/branching_3.png)

```bash
git checkout main
git add --all
git commit --message "fix terrible things"
```

---

## Merge

![branching_4](https://courses.davidl.fr/assets/images/git-101/branching_4.png)

```bash
git checkout main
git diff main..my-feature
git merge my-feature
```

---

## Clean-up

![branching_5](https://courses.davidl.fr/assets/images/git-101/branching_5.png)

```bash
git branch --delete my-feature
```

## We need a remote!

Go to GitHub, create a repo: https://repo.new

```bash
git remote add origin git@github.com:<username>/<project>.git
```

---

## Push

Share the code with your team, and the world

```bash
# Generic command
git push <remote> <branch>

# What we'll use
git push origin main
```

---

## Pull

```bash
# Generic command
git pull <remote> <branch>

# What we'll use
git pull origin main
```
